Curls

A curl is a text status that only you can update.


There is no history, conversation or anything, it's just text that you update.


This is meant to be used with your own tools and interfaces.

Claim a new curl Change the status of a curl you already own View your curls in a dashboard
To view a curl from a browser:
Go to: https://this.website/[curl]

To view a curl from the terminal:
curl https://this.website/[curl]

The response is pure text.
